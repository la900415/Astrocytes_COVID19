# Astrocytes_COVID19
This repository contains most of the proteomic input files and scripts associated to manuscript "Viral infection differentially modulate inflammatory responses in astrocytes of infection-resilient centenarians compared to adults". MS data files can be found in PRIDE repository under the unique identifier PXD075605 (Token: TMMcJba51150). 

# Scripts
Scripts are located in the bin folder. Several packages are required before runnig the scripts. 

# Files
Input files are located in the xlsx folder and corresponds to tables for specific analysis. The quantitative tables (tsv) and dataset.RData file from MS-DAP output have been deposited into the PRIDE repository with the dataset identifier PXD075605 (Token: TMMcJba51150).
